# 计算机学院综合服务平台(CSISP)

## 项目概述

计算机学院综合服务平台(CSISP)是一个为计算机学院师生提供全方位教学管理与服务的综合平台，旨在通过数字化手段提升教学管理效率，优化师生交互体验，实现教学活动的规范化、流程化和智能化。

## 核心功能

- **用户管理**：多角色用户系统（管理员、学生、课代表、学生干部）
- **课程管理**：课程创建、查询、管理及课程与班级、教师的关联
- **考勤管理**：考勤任务发布、签到打卡、统计分析
- **作业管理**：作业发布、提交、评分、统计
- **通知管理**：多级通知发布、阅读状态跟踪
- **教师信息管理**：教师基本信息、授课情况管理

## 技术架构

### 后端技术栈

- **框架**：Koa.js (Node.js)
- **语言**：TypeScript
- **数据库**：PostgreSQL
- **ORM**：Sequelize
- **缓存**：Redis
- **认证**：JWT

### 前端技术栈

- **框架**：Vue 3
- **语言**：TypeScript
- **构建工具**：Vite
- **UI组件库**：Naive-UI
- **状态管理**：Pinia

## 快速开始

### 环境准备

1. 安装 Node.js 21+
2. 安装 pnpm
3. 安装 PostgreSQL 和 Redis

### 开发流程

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```
